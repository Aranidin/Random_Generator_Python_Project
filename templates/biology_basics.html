Biology Basics

Let's start at the beginning! You're probably using this website because you have a DNA sequence and want to learn more about it.

Here are some fundamental concepts to keep in mind when working with DNA sequences: Every living organism is built from the same basic DNA components. DNA consists of four bases—cytosine (C), guanine (G), adenine (A), and thymine (T)—as well as a sugar called deoxyribose, which gives DNA its full name: deoxyribonucleic acid (DNA). These four bases encode all life on Earth.

Now that we have a DNA sequence, we can start uncovering its secrets. Decoding DNA is a straightforward process. First, the sequence is transcribed into RNA. Within each cell, specialized proteins called ribosomes translate the RNA into amino acids, the building blocks of proteins. There are 20 standard amino acids, and they are encoded using triplets of DNA bases.

But how does the ribosome know where a gene starts and ends? Every translation begins with a specific start codon—AUG—which codes for the amino acid methionine. From there, the ribosome moves along the RNA, reading the genetic code in triplets and assembling a chain of amino acids to form a protein.
