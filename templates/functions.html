Functions

Navigation

Use the "Previous" and "Next" buttons to move through the sequence in increments. This makes it easy to browse different regions without losing track of annotations.

Annotated Features

  • Important regions of the DNA (like promoters and signal sequences) are highlighted above the sequence.
  • Each feature is color-coded and labeled with its name and exact position for quick identification.

Feature Table

Below the sequence, you’ll find a feature table listing all annotated elements along with their start and end positions.

This gives a clear, structured overview of the features present in the displayed section.

Viewing Options

  • Linear DNA View: Shows only the annotated features as a simple map, without displaying the full nucleotide sequence.
  • Circular DNA View: Provides a circular map of the sequence, which is especially useful for plasmids and circular genomes.

Primer Functions

  • primer.complement_sequence: Returns the complementary sequence of the given primer.
  • primer.gc_content: Calculates the percentage of guanine (G) and cytosine (C) bases in the primer.
  • primer.tm: Determines the melting temperature (Tm), which is the temperature at which half of the DNA duplex dissociates.
  • primer.GC_clamp: Checks if the first and last two bases of the primer are G or C, improving primer stability.
  • primer.hair_pin: Uses the Rabin-Karp algorithm to detect complementary regions that can form a hairpin structure.
  • primer.double_binding: Detects complementary regions in the sequence that may lead to dimer formation.

Primer Design & Analysis

  • primer_analysis: Generates a table containing all analytical data on the primers.
  • Primer Validation: Checks if the primers exist and computes the length of the amplicon.

Lysis Analysis

Illustrates which restriction enzymes cut the given DNA sequence.

Statistical Analysis

  • gc_content: Determines the proportion of guanine (G) and cytosine (C) bases in the provided DNA sequence.
  • find_orfs: Identifies Open Reading Frames (ORFs) starting with ATG and ending with a stop codon.
  • translate_dna_to_protein: Converts a DNA sequence into its corresponding protein sequence.
  • analyze_protein: Calculates protein properties, including isoelectric point (pI), molecular weight, and amino acid composition.
  • plot_amino_acid_composition: Generates a pie chart of amino acid composition.

Database Functions

  • save_dna_sequence: Stores a DNA sequence and its description in the database.
  • save_gc_content: Calculates and saves the GC content of a sequence.
  • save_orfs: Detects ORFs and stores their start and stop positions.
  • save_protein_sequence: Stores the translated protein sequence.
  • save_protein_analysis: Saves protein properties like molecular weight and isoelectric point.
  • save_visualization: Stores generated visualization files for reference.
  • close_db: Ensures all transactions are committed and closes the database connection.
